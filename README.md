# Full-Stack-Car-Dealer-Capstone-Python-JS

This is the comprehensive capstone application index and management panel. It acts as an interactive client portal and administrative dashboard written in modern React, JavaScript/TypeScript, and Tailwind CSS. The complete enterprise application represents a full-stack, distributed microservices architecture.

## 🛠️ Technology Stack Breakdown

### Frontend
- **React 19 & Vite** - Next-generation compiler, state-management hooks, dynamic view routing, and hot module representations.
- **Tailwind CSS** - Modern utility-first spacing, grid alignments, and custom light/dark styling tokens.
- **Motion** - Fluid page layout transitions and hover state micro-interactions.

### Backend & Microservices
- **Django Framework (Python 5.0)** - Core administrative framework managing authentication, user session security, and state proxy pipelines.
- **Flask (Python Microservice)** - Autonomous natural language processing service handling live sentiment classification for reviews.
- **Node.js** - High-concurrency reverse proxy and asset handler running intermediate gateway routes.

### Databases
- **MongoDB (Cloud Cluster)** - Unstructured document database facilitating lightning-fast customer reviews query lookups.
- **SQLite (Local Relational DB)** - Light, serverless internal SQL database managing strict relational car makes, models, and administrative records.

### DevOps & Deployment Pipeline
- **Docker** - Self-contained multi-stage containerization defining repeatable client/server running environments.
- **Kubernetes (K8s Clusters)** - High-availability service orchestration, load balancing, pod scaling, and health check routines.
- **IBM Cloud Code Engine** - Production-grade serverless cloud platform hosting containerized cluster endpoints safely.

## 💡 How to Use
1. **Explore Dealerships**: Filter dealerships based on state (Kansas, Texas, California, Florida) or search directly by city or location name.
2. **Review Client Feedback**: Select any dealership to drill down into active reviews. Reviews are classified dynamically via sentiment analytics.
3. **Submit Reviews**: Use the live "Write a Review" form to enter new owner feedback with stars, cars, and purchase histories.
4. **Django Site Administration**: Log in via the **Django Admin** tab using `admin` / `rootpassword`. Perform full-scale CRUD on car makes and models, switch UI color themes, and view live sqlite transaction logging histories.
5. **Inspect Deliverables**: Access the **Deliverables Console** to view simulated logs from actual Django REST backend microservices.
